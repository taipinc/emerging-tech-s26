# Workshop: Agentic (AI) Coding (w/ Goole Gemini)

In this workshop, we will explore the concept of agentic coding using Google Gemini. Agentic coding refers to the ability of AI systems to autonomously generate code based on user prompts and requirements. We will learn how to leverage Google Gemini's capabilities to create discuss project specs, generate code, and build simple applications.

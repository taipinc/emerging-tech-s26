# Workshop: p5.js + VS Code (and Github)

In this workshop, we will learn how to set up a p5.js project using Visual Studio Code (VS Code) and GitHub. We will cover how to install VS Code and the p5.js plugin, and connect it to a new Github Education account.

Or the short version:
1. Download and install [VS Code](https://code.visualstudio.com/) (AKA Visual Studio Code)
2. In VS Code, install the [p5.js plugin](https://marketplace.visualstudio.com/items?itemName=samplavigne.p5-vscode)
3. Sign up for an account at [Github](https://github.com/) using your private email address
4. Apply for [Github Education](https://github.com/education) by adding your school email address

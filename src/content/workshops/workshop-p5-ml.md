# Workshop: p5.js + ml5.js (Machine Learning!)

In this workshop, we will explore how to use machine learning in creative coding projects using p5.js and the [ml5.js](https://ml5js.org/) library. We will cover the basics of machine learning and how to use pre-trained models for image classification, pose estimation, and more.

# Workshop: cables.gl

[cables](https://cables.gl/) is an online tool which allows people to create interactive visual web content without having to type a line of code. It's design and UI make it accessible to all users regardless of experience level.

Rather than writing code, you'll be interfacing with connectors (cables) between different components, building the logics and operations of your app.
